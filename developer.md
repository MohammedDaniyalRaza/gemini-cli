# Developer Guide

Welcome, aspiring developer! This guide is for you to get started with contributing to this project.

## Getting Started

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/MohammedDaniyalRaza/gemini-cli.git
    ```
2.  **Install dependencies:**
    ```bash
    npm install
    ```
3.  **Run the project:**
    ```bash
    npm start
    ```

## Contributing

We welcome contributions of all kinds! Please refer to our `CONTRIBUTING.md` (if available) for more details.

Happy coding!
